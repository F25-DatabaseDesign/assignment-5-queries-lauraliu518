1) How many containers of antibiotics are currently available?
Query:
SELECT quantityOnHand 
FROM item 
WHERE itemDescription = "bottle of antibiotics";
Answer: 100

2) Which volunteer(s), if any, have phone numbers that do not start with the number 2 and
whose last name is not Jones. Query should retrieve names rather than ids.
Query: 
SELECT volunteerName 
FROM volunteer 
WHERE volunteerTelephone NOT LIKE "2%" AND volunteerName NOT LIKE "% Jones";
Answer: Gene Lewin

3) Which volunteer(s) are working on transporting tasks? Query should retrieve names
rather than ids.
Query:
SELECT v.volunteerName 
FROM volunteer AS v
JOIN assignment AS a ON a.volunteerId = v.volunteerId
JOIN task AS t ON t.taskCode = a.taskCode
JOIN task_type AS tt ON tt.taskTypeId = t.taskTypeId
WHERE tt.taskTypeName = "transporting";
Answer: George Brewer

4) Which task(s) have yet to be assigned to any volunteers (provide task descriptions, not
the codes?
Query:
SELECT t.taskDescription
FROM task AS t
LEFT JOIN assignment AS a ON a.taskCode = t.taskCode
WHERE a.volunteerId IS NULL;
Answer: 'Prepare 100 food packages', 'Take packages to the warehouse'

5) Which type(s) of package contain some kind of bottle? 
Query:
SELECT DISTINCT type.packageTypeName 
FROM package_type AS type
JOIN package AS p ON p.packageTypeId = type.packageTypeId
JOIN package_contents AS content ON content.packageId = p.packageId
JOIN item AS i on i.itemId = content.itemId
WHERE i.itemDescription LIKE "%bottle%" OR i.itemDescription LIKE "%Bottle%";
Answer: 'food and water' (Assumption: "Which type" refers to the type in words and not in type id.)

7) Which task(s) are assigned to volunteer(s) that live in New Jersey (NJ)? Answer should
have the task description and not the task ids.
Query:
SELECT DISTINCT t.taskDescription 
FROM task AS t
JOIN assignment AS a ON a.taskCode = t.taskCode
JOIN volunteer AS v ON v.volunteerId = a.volunteerId
WHERE v.volunteerAddress LIKE "%NJ%";
Answer: 'Prepare 20 children's packages', 'Prepare 100 water packages', 'Prepare 5,000 packages'

9) Which volunteers have been assigned to tasks that include packing spam? Answer

10) Which item(s) (if any) have a total value of exactly 100 in one package? Answer should
be item descriptions.
Query:
SELECT i.itemDescription 
FROM item AS i 
JOIN package_contents AS content ON content.itemId = i.itemId
WHERE (content.itemQuantity * i.itemValue) = 100.00; 
Answer: Baby formula

11) How many volunteers are assigned to tasks with each different status? The answer
should show each different status and the number of volunteers sorted from highest to
lowest)
Query:
SELECT ts.taskStatusName, COUNT(DISTINCT a.volunteerId)
FROM task_status AS ts
JOIN task AS t ON t.taskStatusId = ts.taskStatusId
JOIN assignment AS a ON a.taskCode = t.taskCode
GROUP BY ts.taskStatusName
ORDER BY COUNT(DISTINCT a.volunteerId) DESC;
Answer:
'open','4'
'closed','3'
'ongoing','2'
'pending','1'

12) Which task creates the heaviest set of packages and what is the weight? Show both the
taskCode and the weight (You should be able to do this without using any sub-queries).
Query:
SELECT t.taskCode, SUM(p.packageWeight) AS weight
FROM task AS t
JOIN package AS p ON p.taskCode = t.taskCode
GROUP BY t.taskCode
ORDER BY weight DESC
LIMIT 1;
Answer:
'106','308'

13) How many tasks are there that do not have a type of “packing”?
Query:
SELECT COUNT(DISTINCT t.taskCode) AS taskCount
FROM task AS t
JOIN task_type AS type ON type.taskTypeId = t.taskTypeId
WHERE type.taskTypeName != "packing";
Answer: 3

14) Of those items that have been packed, which item (or items) were touched by fewer
than 3 volunteers? Answer should be item descriptions.
Query:
SELECT i.itemDescription 
FROM item AS i
JOIN package_contents AS content ON content.itemId = i.itemId
JOIN package AS p ON p.packageID = content.packageId
JOIN task AS t ON t.taskCode = p.taskCode
JOIN task_status AS status ON status.taskStatusId = t.taskStatusId
JOIN assignment AS a ON a.taskCode = t.taskCode
JOIN volunteer AS v ON v.volunteerId = a.volunteerId
WHERE status.taskStatusName = "closed" 
GROUP BY i.itemId
HAVING COUNT(DISTINCT v.volunteerId) < 3;
Answer: 'Baby formula', 'men's coat' (Assumption: I interpreted "have been packed" as referring to the completed tasks with status = "closed".)

15) Which packages have a total value of more than 100? Show the packageIds and their
value sorted from lowest to highest.
Query:
SELECT p.packageId, SUM(i.itemValue * content.itemQuantity) AS tv
FROM package AS p
JOIN package_contents AS content ON content.packageId = p.packageId
JOIN item AS i ON i.itemId = content.itemId
GROUP BY p.packageId
HAVING tv > 100
ORDER BY tv ASC;
Answer:
'6','150'
'2','151'
'4','350'
'10','750'
'5','1420'
